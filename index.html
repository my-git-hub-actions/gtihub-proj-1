Because the basics are important!

Basic Git Commands

git init

Initializes a new Git repository

git add <file>

Stages changes for commit

git commit -m "message"

Commits staged changes with a descriptive message

Common Git Operations

git status

Shows the status of your working directory

git log

Displays commit history

git branch

Lists all local branches

Branch Operations

git branch

Lists all local branches

git checkout -b <branch-name>

Creates and switches to a new branch

git merge <branch-name>

Merges specified branch into current branch

Remote Operations

git remote add origin <repo-url>

Connects local repository to remote repository

git push -u origin main

Pushes commits to remote repository

git pull origin main

Fetches and merges changes from remote repository
